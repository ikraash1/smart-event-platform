# Screenshots

This folder contains application screenshots for the README and project documentation.

## How to Add Screenshots

1. Run the application locally (see Installation Guide in `docs/` or `../docs/INSTALLATION_GUIDE.md`)
2. Take screenshots of the following key pages:
   - Home page (hero + trending events)
   - Event listing page with search/filters
   - Event details page with booking panel
   - My Tickets page showing QR codes
   - QR Scanner page (camera active)
   - Organizer Dashboard (stat cards + charts)
   - Admin Dashboard (platform overview)
   - Recommendations page (AI picks with match %)
   - Register / Login pages

## Recommended File Names

```
home.png
events-listing.png
event-details.png
booking-success.png
my-tickets-qr.png
qr-scanner.png

admin-dashboard.png
recommendations.png
login.png
register.png
```

## Screenshot Tips

- Use a browser window sized to at least 1280×800 for desktop screenshots
- For mobile, use Chrome DevTools Device Toolbar (F12 → mobile icon) at 375×812 (iPhone X)
- Include both light and dark state screenshots where visible
- Capture the QR scanner with a real QR code visible in the preview
